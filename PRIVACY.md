# Privacy policy

Effective date: 2026-07-17

MDViewer is a local-only document conversion application. The application does not collect
analytics, telemetry, advertising identifiers, account information or document contents, and it
does not send converted documents or OCR data to a remote service.

Files selected or printed by the user are processed on the user's device. OCR uses Apple Vision on
macOS, Windows Media OCR on Windows, or the packaged/system Tesseract runtime on Linux. Temporary
files and local print jobs are bounded and removed by the lifecycle described in the product
documentation.

MDViewer does not operate a cloud service and the maintainers do not receive personal data from the
application. If a user deliberately follows an external link, downloads a release from GitHub, or
submits a GitHub issue or private security advisory, that interaction is governed by the privacy
policy of the service the user chose to contact.

Changes to this policy will be published in this repository. Privacy questions may be opened as a
public GitHub issue when they contain no sensitive information. Security-sensitive questions must
use the private reporting channel in [SECURITY.md](SECURITY.md).
